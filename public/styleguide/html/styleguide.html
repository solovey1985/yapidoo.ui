    Tokens

    Brand Colors

    • $color-primary
      #ff8219
    • $color-secondary
      #ffe81e
      Color is used for building of gradient.
    To add to these items, use Sass variables that start with $color-brand- in ./src/assets/styles/_colors.scss

    Gradients

    • $dark-gradient
      linear-gradient(180deg, #e5e5e5 -418.4%, #26282e 100%);
      Errors, warnings, etc.
    • $color-radial-gradient
      radial-gradient(#ffffff 20.96%, #e8e4e4 79.75%, #e8e4e4 91.78%);
      Buttons, links etc.
    • $color-orange-gradient
      linear-gradient(to bottom, #ff8219 0%, #ffe81e 100%);
      Buttons, links pressed state, etc.
    • $gradient-primary-hover
      linear-gradient(0deg, rgba(0, 0, 0, 0.2) 0%, rgba(0, 0, 0, 0.2) 100%), linear-gradient(232deg, #ff8219 31.4%, #ffe81e 122.19%);
      Buttons, links hover state, etc.
    • $gradient-default-hover
      linear-gradient(0deg, rgba(255, 130, 25, 0.2) 0%, rgba(255, 130, 25, 0.2) 100%), #f5f6f8;
      Buttons, links hover state, etc.
    • $gradient-default-pressed
      linear-gradient(0deg, rgba(0, 0, 0, 0.2) 0%, rgba(0, 0, 0, 0.2) 100%), linear-gradient(232deg, #ff8219 31.4%, #ffe81e 122.19%);
      Buttons, links hover state, etc.
    To add to these items, use Sass variables that start with $color-utility- in ./source/css/scss/abstracts/_variables.scss
